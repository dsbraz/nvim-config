# nvim-config

Personal Neovim configuration using `lazy.nvim`.

## Features

- Catppuccin theme
- File tree with `nvim-tree`
- Fuzzy finding with Telescope
- Harpoon navigation
- LSP management with Mason
- Completion with `nvim-cmp`
- Formatting with Conform
- Git signs
- Diagnostics with Trouble
- Keybinding hints with Which Key

## Structure

```text
init.lua
lua/config/
lua/plugins/
```
